## 1.查看md文档的方式：

在vscode中点击最上边的终端-新建终端，在终端中输入命令

```bash

bundle exec jekyll serve

```

输入命令后的输出如下：

![alt text]({{site.baseurl}}/img/image.png)

一定要能正常显示网址，并打开该网址：localhost:4000，可以看到网站的显示效果图。

**特别注意:如左下角有类似弹窗，应点击refresh按钮更新页面内容方可看到新增内容**

![alt text]({{site.baseurl}}/img/image-1.png)

## 2.添加md文档的方式：

在vscode打开的终端中输入以下命令：

```bash

rake post title="文章主标题" subtitle="文章副标题"

```

其后，可以在_posts文件夹下看到新增的md文档并打开文档。显示如下为正常：

![alt text]({{site.baseurl}}/img/image-2.png)

**可使用latex格式公式**

最后刷新本地端页面即可看到新post的文章

## 3. 上传github注意事项：

1. 将_posts文件夹下的所有image移动到img文件夹下
2. 将所有的文件路径作如下修改：**仅在vscode中查看无需修改，直接粘贴即可**
   ![alt text]({{site.baseurl}}/img/image-3.png)
   ![alt text]({{site.baseurl}}/img/image-4.png)

3. 选择左侧的源代码管理， 点击暂存所有更改，输入commit信息，点击提交，点击同步更改。
4. 待github页面的这个位置显示对号时即上传网页成功
   ![alt text]({{site.baseurl}}/img/image-9.png)

   **注意图片名称，千万别冲突了**
